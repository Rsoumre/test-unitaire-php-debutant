# Test Unitaire PHP Débutant

## Initialisation du projet PHP avec Composer
Création du dossier `test-unitaire-php-debutant` et initialisation d'un projet PHP avec la commande `composer init`. Cette commande génère automatiquement un fichier`composer.json` qui contient la configuration du projet.

![Initialisation du projet avec Composer]
![Initialisation du projet avec Composer](<images/Capture d’écran du 2026-06-08 13-29-06.png>)

## Installation de PHPUnit
Installation de PHPUnit en utilisant Composer avec la commande `composer require --dev phpunit/phpunit`. Cette commande ajoute PHPUnit en tant que dépendance de développement dans le projet.

![Installation de PHPUnit avec Composer]
![Installation de PHPUnit avec Composer](<images/Capture d’écran du 2026-06-08 13-40-20.png>)

## Lancement des tests

Lancement des tests avec la commande `./vendor/bin/phpunit tests/MathTest.php`. Le `.` indique que le test a passé avec succès.
